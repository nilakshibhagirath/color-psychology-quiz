# 🎨 Color Psychology Quiz

A fun and interactive web application that determines your personality color based on your responses to a series of carefully designed questions. The quiz uses color psychology concepts to provide insights into personality traits, emotions, and behavioral tendencies.

## 🚀 Features

* Interactive personality quiz
* Dynamic question and answer system
* Instant result generation
* Color-based personality analysis
* Responsive design for desktop and mobile devices
* Clean and user-friendly interface

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 🎯 How It Works

1. Start the quiz and answer all questions honestly.
2. Each answer contributes points toward a specific color category.
3. Once all questions are completed, the application calculates your dominant color.
4. Your personality color and its meaning are displayed instantly.
